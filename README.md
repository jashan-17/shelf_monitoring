# Supermarket Shelf Stock Monitoring System

This project is a computer vision system that classifies supermarket shelf images into stock levels:

- Empty
- Low
- Medium
- Full

## Objective
Build and train a Convolutional Neural Network (CNN) from scratch (no transfer learning) to automatically monitor shelf stock levels using images.

## Tech Stack
- Python
- TensorFlow / Keras
- OpenCV

## Features
- Custom CNN model built from scratch
- Image classification into 4 stock levels
- Data preprocessing using OpenCV
- Model training and evaluation
- Real-world use case for retail inventory monitoring

## Project Structure
- dataset/ → training, validation, testing images
- src/ → core code
- models/ → saved models
- notebooks/ → experiments
